Belajar CSS

Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan dokumen web lain,yaitu HTML. bebrapa contoh pemanfaatan CSS dan HTML adalah untuk mengatur warna latar belakang, gaya huruf,jarak antar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberdaannya tidak menggantikan HTML melainkan sebagai pelengkap agar tampilan dokumen HTML menjadi lebih indah dan dinamis. berikut ini adalah contoh dokumen CSS

Bagaimana Menggunkaan CSS?

ada 3 cara menggunkaan CSS:

  1. inline Styles: menuliskan kode CSS langsung pada tag yang akan diubah atau di atur style-nya
  2. internal/Embedded Styles: menuliskan kode CSS pada bagian header dokumen HTML
  3. External/Linked Styles: menuliskan kode CSS pada file terpisah dan di panggil melalui tag link dari dokumen HTMl
